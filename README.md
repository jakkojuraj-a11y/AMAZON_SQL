
# 📘 Amazon SQL Interview Questions

## 🌍 Introduction
This repository provides a **comprehensive collection of SQL interview questions frequently asked at Amazon**. It is designed to help candidates strengthen their SQL skills, prepare for technical interviews, and practice real-world query scenarios.

The content emphasizes **clarity, reproducibility, and practical application**, making it suitable for professionals, students, and job seekers worldwide.

---

## 📂 Structure
The document is organized into **20 core SQL problems**, each with:
- **Problem statement** – clear and concise  
- **SQL query solution** – written in ANSI SQL  
- **Explanation** – step-by-step reasoning  

---

## 🗒️ Topics Covered
- Ranking and aggregation (`MAX`, `AVG`, `RANK`, `ROW_NUMBER`)  
- Window functions (`LAG`, `SUM OVER`)  
- Self-joins and subqueries  
- Data cleaning (duplicates, gaps, sequences)  
- Pivoting and transformations  
- Date and time operations  
- Advanced analytics (median, cumulative percentage, consecutive sequences)  

---

## 🛠️ Usage Guidelines
1. **Set up sample tables** using the provided schema (`employees`, `sales`, `dates`, etc.).  
2. **Run queries step by step** to understand logic and verify outputs.  
3. **Experiment with variations** to handle edge cases (NULLs, duplicates, missing data).  
4. **Adapt queries** to your preferred SQL dialect (MySQL, PostgreSQL, SQL Server, Oracle).

## 🎯 Target Audience
- **Job Seekers** preparing for SQL-heavy interviews at Amazon and other tech companies.

## 📖 Best Practices
- Queries are written in **ANSI SQL** for portability.  
- Adjust syntax for vendor-specific implementations (e.g., `LIMIT` vs. `TOP`).  
- Focus on **conceptual clarity** rather than database-specific optimizations.
